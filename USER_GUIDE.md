# World Lens 利用ガイド

World Lens は、世界各国の重要な変化を地図・国別ブリーフ・出典付きで確認するためのデモアプリです。
報道の少ない国の変化に「受動的に出会う」ことを重視した並び順・表示になっています。

## まず見る場所

- **3分で読む今日の世界**: 各国の結果をAIが統合したダイジェスト。まずここで全体像を掴みます。
- **横断テーマ**: 複数の国にまたがる共通の動き(通貨安、気候適応など)。国名をクリックするとその国へ移動します。
- **今日のスポットライト**: 報道の少ない国を日替わりで1か国紹介します。
- **世界マップ**: どの地域・国で重要な変化が起きているかを俯瞰します。地域タブでズームします。
- **今日のインテリジェンス・ブリーフ**: 発見スコア順に国別トピックを確認します。
- **選択トピック詳細**: 要約、重要な理由、現地文脈、根拠URLを確認します。

## 基本操作

1. 画面上部の地域タブで、見たい地域を選びます(地図がその地域にズームします)。
2. 「注目トピック」でテーマを選ぶと、**表示中の一覧・地図・分布がそのカテゴリで絞り込まれます**。リサーチ開始時の調査テーマにもなります。
3. APIキー設定済みなら「AIリサーチ開始」を押します。実行中は完了した国から順に画面へ反映されます。
4. **ブリーフのニュースをクリックすると、ポップアップで詳細（要約・なぜ重要か・現地文脈・出典）が開きます**。Escキー・✕・背景クリックで閉じられます。
5. **地図のピンをクリックすると、ブリーフがその国のニュース一覧に切り替わります**。「← 全体ブリーフへ」ボタン（または地域タブの切り替え）で通常のブリーフに戻ります。
6. 調査カバレッジ、シグナル分布、横断テーマの国名から選んだ場合は、右下の「選択トピック詳細」欄に表示されます。

## 画面の見方

- **並び順 = 発見スコア**: 重要度 × 露出補正(主要国 1.0 / 地域国 1.2 / 報道が少ない国 1.5)。さらにブリーフ上位には「報道が少ない国」が最低2件入るよう補正されます(該当データがある場合)。
- **「報道が少ない国」バッジ**: メディア露出が少ないため加点表示されている国です。
- **「新規」「継続」バッジ**: 前回のリサーチ結果と比べて、新しく出てきたトピックか、続いているトピックかを示します。
- **赤いシグナル**: 重要度が高い変化(85以上)。**オレンジ**: 中程度(70-84)。**緑**: 比較的低い変化・継続動向です。
- **調査カバレッジ**: 全対象国の取得状況。灰色は「表示条件に合う情報なし」、赤枠は「取得失敗」です。失敗した国も過去の成功データがあれば表示は残ります。
- **ソース多様性**: 表示中トピックの出典の言語分布と現地語ソース比率。情報の偏りを数値で確認できます。
- **シグナル分布**: 横軸が重要度スコア(実データ)。70 / 85 が色の閾値です。
- **今週の探索**(左下): この1週間で閲覧した国数と、報道の少ない国への接触数。あなたの世界カバレッジの目安です。
- **データ鮮度**: 画面上部に最終リサーチからの経過時間を表示します。24時間以上経つと注意色になります。

## 受動的な自動更新

- `.env` の `RESEARCH_AUTO_TIME=07:00` のように設定すると、サーバー起動中は毎日その時刻に自動リサーチが走ります。
- サーバーを常時起動しない場合は、`npm run research` をOSのタスクスケジューラに登録してください。
- 朝アプリを開くだけで「今日の世界」が用意されている状態を目指した機能です。

## 注意点

- APIキー未設定の場合、実リサーチは実行できません。画面にはデモデータが表示されます。
- 実リサーチ結果はAIによる要約です。重要な判断に使う場合は、必ず出典URLを確認してください。
- 重要度・信頼度はAIの自己評価であり、国どうしの厳密な比較には限界があります(採点基準をプロンプトで統一しています)。
- PoC版の対象国は代表20か国です。全193か国を網羅する本番版ではありません。
- 1国あたりの主要トピックは最大3件です。
- リサーチには時間とAPI利用料金がかかります(並列実行数は `RESEARCH_CONCURRENCY` で調整)。

## このアプリの目的

普段のニュースやSNSでは見落としやすい国・地域の変化に自然に出会い、世界情勢をより横断的に理解することを目的としています。
そのため「重要度が高い順」ではなく「重要度 × 露出の少なさ」で並べ、どの国を取得できて どの国が欠けているかも隠さず表示します。
